# CLI Trivia Game

A command-line trivia game built with JavaScript. This application challenges users with time-limited questions and tracks their score.

## Features
- **Sequential Questions:** View and answer trivia questions one by one.
- **Timed Gameplay:** A 10-second timer per question to keep the game engaging.
- **Immediate Feedback:** Instant confirmation if your answer is correct or incorrect.
- **Score Tracking:** Keeps track of your progress throughout the session.

## Prerequisites
- [Node.js](https://nodejs.org/) installed on your machine.

## How to Run
1. Clone this repository to your local machine.
2. Open your terminal in the project folder.
3. Run the following command to start the game:
   ```bash
   node index.js
